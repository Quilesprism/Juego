"Carrera" es juego de competición en el que el usuario debe ganar el puntaje mas alto que pueda.
abra el juego, despues de eso aparecera una pantalla como si fuera de perdida, tranquilo, solo oprima
la primera opción y así inciara el juego.
El juego tiene varios niveles, cuando llegue a 50 puntos el nivel aumentara, poco a poco con
cada nivel observara que el juego irá mas rapido. 
cuando el carro choca con otro carro, el juego se reinicia y se pierde el puntaje que tenia anteriormete.
